## 1AI — AI for Every 1

**1AI** is an AI-powered search experience built into the Blue-Star project. It features **Re-Search**, an intelligent search engine that uses LLM (Large Language Model) technology to understand natural-language questions and provide conversational, context-aware answers.

### Features

- **Re-Search Engine** — LLM-powered search that understands intent, not just keywords
- **AI for Every 1** — Accessible AI designed to serve every individual
- **Conversational Interface** — Ask questions naturally and receive structured answers
- **Knowledge Base** — Covers 1AI, Re-Search, G.O.S.©, QR Store©, G.L.S.©, and Blue-Star

### Getting Started

Open `index.html` in a web browser to use the 1AI Re-Search engine. You can also deploy it via GitHub Pages.
